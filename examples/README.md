```bash

```

### 方法2: 使用快速分析

```bash
# 60秒快照（单智能体）
/ai-trading:quick 600519
```

### 方法3: 单维度深度分析

```bash
# 只分析技术面
/ai-trading:market 600519

# 只分析基本面
/ai-trading:fundamentals 600519

# 只分析新闻面
/ai-trading:news 600519

# 只分析情绪面
/ai-trading:sentiment 600519

# A股专属视角
/ai-trading:china 600519
```

## 📊 报告结构说明

所有完整分析报告遵循以下结构：

```
1. 📋 执行摘要
   - 投资建议明确（买入/持有/卖出）
   - 置信度评分
   - 目标价位
   - 核心投资逻辑（5句话以内）

2. 📈 技术面分析
   - 趋势判断
   - 技术指标（RSI, MACD, 均线系统）
   - 支撑/压力位
   - 交易策略建议

3. 💰 基本面分析
   - 估值水平（PE, PB, PS）
   - 盈利能力（ROE, ROA, 利润率）
   - 现金流健康度
   - DCF估值

4. 📰 新闻面分析
   - 近期重要新闻
   - 宏观环境
   - 媒体情绪

5. 😊 情绪面分析
   - 社交媒体情绪
   - 机构动向
   - 资金流向

6. 🇨🇳 A股特色视角（仅A股）
   - 政策与监管
   - 资金面分析
   - 涨跌停与异动
   - A股特有风险

7. 🐂🐻 多空辩论
   - 多方观点
   - 空方观点
   - 研究经理综合判断

8. 💼 交易员提案
   - 具体操作建议
   - 建仓策略
   - 止盈止损位

9. 🔥🛡️⚖️ 风险三方辩论
   - 激进派
   - 保守派
   - 中性派

10. 🎯 风险经理最终决策
    - 最终投资建议
    - 详细操作方案
    - 风险提示
    - 监控点
```

## 💡 报告使用技巧

### 快速阅读法（2分钟）

只看这3个部分：
1. **执行摘要** - 了解核心结论
2. **风险经理最终决策** - 获取具体操作方案
3. **风险提示** - 识别关键风险

### 深度研读法（15分钟）

按顺序阅读全部章节，重点关注：
- 多空辩论中的**关键分歧点**
- 基本面分析中的**估值合理性**
- 风险三方辩论中的**不同风险偏好建议**

### 对比研究法

生成同行业多家公司报告，横向对比：
```bash
/ai-trading:analyze 600519  # 贵州茅台
/ai-trading:analyze 000858  # 五粮液
/ai-trading:analyze 000568  # 泸州老窖
```

## ⚠️ 使用报告的注意事项

1. **报告时效性**
   - 行情数据：准实时（数据源延迟<5分钟）
   - 财报数据：以最新披露为准
   - 新闻数据：最近30天
   - 情绪数据：最近7天

2. **数据精确性**
   - 所有估值计算使用 `decimal.Decimal`，避免浮点误差
   - 关键数据建议人工交叉验证
   - AI生成内容可能存在解读偏差

3. **投资决策**
   - 报告仅供参考，不构成投资建议
   - 必须结合自身风险偏好和资金状况
   - 强烈建议咨询专业财务顾问
