# 快速开始 - AI Trading Skills

## 🎯 30 秒安装

```bash
# 方式 1：从 GitHub 安装（推荐，需要先发布）
npx skills add Yafan-Yang/ai-trading

# 方式 2：从本地安装（开发/测试）
git clone <your-repo-url> ai-trading
cd ai-trading
npx skills add .
```

安装后，在任何支持 skills.sh 的智能体中使用。

## 📱 支持的智能体

✅ **Claude Code** - `/ai-trading:analyze 600519`  
✅ **Codex** - 自然语言："使用 ai-trading analyze 分析腾讯"  
✅ **Cursor** - 直接对话："用 ai-trading 分析 AAPL"  
✅ **OpenCode, Cline, Windsurf** 等 60+ 智能体

## 💡 快速示例

### 完整分析（推荐）
```
使用 ai-trading analyze 分析贵州茅台 (600519)
```

**输出**：
- 📈 技术面分析（趋势、指标）
- 💰 基本面分析（财务、估值）
- 📰 新闻面分析（最新动态）
- 🌡️ 情绪面分析（市场情绪）
- ⚔️ 多空辩论（看涨 vs 看跌）
- 🛡️ 风险评估（三方辩论）
- 📊 **最终决策**：买入/持有/卖出 + 目标价

### 60秒快照
```
使用 ai-trading quick 查看 AAPL 快照
```

### 单维度分析
```
用 ai-trading market 做 0700.HK 的技术分析
用 ai-trading fundamentals 看茅台的基本面
用 ai-trading news 获取特斯拉最新新闻
```

## 🌍 支持的市场

| 市场 | 代码格式 | 示例 |
|------|---------|------|
| **A股** | 6位数字 | `600519` (茅台), `000001` (平安) |
| **港股** | 4位数字.HK | `0700.HK` (腾讯), `9988.HK` (阿里) |
| **美股** | 股票代码 | `AAPL`, `TSLA`, `NVDA` |

## 🚀 12个可用技能

大多数场景从主入口开始即可，无需记住全部命令。

**🎯 主入口（先用这两个）**

| Skill | 用途 | 时长 |
|-------|------|------|
| `analyze` | 完整多智能体分析流水线（自动编排所有分析师+辩论） | ~5分钟 |
| `quick` | 60秒快照 | ~1分钟 |

**🔬 流程扩展（选股前 / 持有后）**

| Skill | 用途 | 时长 |
|-------|------|------|
| `industry-scan` | 行业扫描，初筛候选清单（选股前，仅A股） | ~2分钟 |
| `moat-analysis` | 护城河四维评分 | ~3分钟 |
| `alert-monitor` | 持仓价格与基本面背离诊断（持有后） | ~2分钟 |

**🧩 单维度钻取**

| Skill | 用途 | 时长 |
|-------|------|------|
| `market` | 技术面（趋势、指标） | ~2分钟 |
| `fundamentals` | 基本面（财务、估值、DCF） | ~2分钟 |
| `news` | 新闻面（最新动态） | ~2分钟 |
| `sentiment` | 情绪面（社交媒体） | ~2分钟 |
| `china` | A股专属视角 | ~2分钟 |
| `debate` | 多空辩论 | ~3分钟 |
| `risk-panel` | 风险三方辩论 | ~3分钟 |

## 🔧 常见问题

### Q: 数据从哪里来？
A: 
- A股/港股：akshare（免费）
- 美股：yfinance（免费）
- 无需 API key

### Q: 分析准确吗？
A: 
- ✅ 数据是真实的（实时拉取）
- ✅ 估值计算经过 Decimal 精确验证
- ⚠️ AI 分析仅供参考，不构成投资建议

### Q: 可以离线使用吗？
A: 不可以，需要联网获取实时数据。

### Q: 支持期货/期权吗？
A: 目前仅支持股票。

### Q: 如何更新？
```bash
npx skills update ai-trading
```

### Q: 如何卸载？
```bash
npx skills remove ai-trading
```
