你将**独立扮演**激进、保守、中性三位风险分析师与风险经理，对 **$ARGUMENTS** 的交易决策做风险辩论并拍板。

## 第一步：确定待评估的交易提案
- 若参数中已给出提案（如“买入/持有/卖出 + 目标价”），以此为评估对象。
- 若只给了代码，先获取真实数据并形成一个初步交易提案：
```bash
__AITRADING_HOME__/.venv/bin/python __AITRADING_HOME__/tools/market_data.py $ARGUMENTS --json
__AITRADING_HOME__/.venv/bin/python __AITRADING_HOME__/tools/fundamentals.py $ARGUMENTS --json
```

## 第二步：三方辩论（轮流发言，互相反驳）

### 🔥 激进分析师
倡导高回报、高风险策略，质疑保守与中性立场过于谨慎，论证承担风险的收益。

### 🛡️ 保守分析师
优先资本保全、降低波动，指出激进/中性观点忽视的下行风险与可持续性问题。

### ⚖️ 中性分析师
提供平衡视角，指出激进过于乐观、保守过于谨慎之处，主张适度风险策略。

## 第三步：🏛️ 风险经理拍板
评估三方辩论，产出**明确且可操作**的最终建议：
1. **总结**各方最强论点
2. **理由**：用辩论中的具体论据支撑
3. **完善交易计划**：在原提案基础上按风险洞察调整（仓位、止损、分批）
4. **最终决策**：买入 / 持有 / 卖出 + 目标价 + 置信度(0-1) + 风险评分(0-1)

## 输出格式（中文）
```
## 🔥 激进观点
## 🛡️ 保守观点
## ⚖️ 中性观点
## 🏛️ 风险经理最终决策（建议 / 目标价 / 置信度 / 风险评分 / 完善后的计划）
```

> 本分析仅用于研究和教育目的，不构成投资建议。投资有风险，决策需谨慎。
